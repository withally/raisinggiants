A reusable component for showing partial PDF chapter previews with blur/fade overlay.

Props:
```ts
interface TeaserSectionProps {
  title: string           // e.g., "Foundational Patterns", "Watchouts"
  chapterLabel: string    // e.g., "In your full guide: Foundational Patterns"
  visibleContent: string  // First theme — shown clearly
  blurredContent: string[] // Remaining themes — shown faded/blurred
}
```

Layout:
1. **Chapter label**: Small uppercase text `text-xs uppercase tracking-wider text-stone-400 mb-1` — "In your full guide"
2. **Section title**: `text-xl font-semibold text-stone-800 mb-3`
3. **Visible content**: First theme displayed clearly in `text-stone-700 leading-relaxed`
4. **Blurred content zone** (relative container):
   - Remaining themes rendered with decreasing opacity: first blurred item at `opacity-30`, second at `opacity-15`, third at `opacity-10` — all with `select-none` to prevent copying
   - Gradient overlay: `absolute inset-0 bg-gradient-to-b from-transparent to-amber-50 pointer-events-none` (fades to page background)
   - Lock text at bottom: "Unlock in your complete blueprint" in `text-stone-500 text-sm text-center`

This is CSS-only blur/fade, no JavaScript. Consistent with RESEARCH.md Section 6 approach but using gradient fade (cleaner than CSS blur on mobile).
