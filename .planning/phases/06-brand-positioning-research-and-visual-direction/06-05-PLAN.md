Phase B (Component-Level — Medium Risk):
- Landing page section copy and layout refinements
- Quiz UI color/type adjustments

- Illustration style updates (if minor evolution)

Phase C (Asset-Level — Higher Effort, if needed):
- New illustrations (if system direction changes significantly)
