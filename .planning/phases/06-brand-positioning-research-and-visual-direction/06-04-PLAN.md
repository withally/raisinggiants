**CRITICAL SYNTHESIS — resolve the old/new conflict:**

---

**Part 1: Design Philosophy (short — 1 page max)**

State the synthesis formula and what it means in practice:
- "#5 Neo-Minimalism + #6 Emotion-Led, seasoned with #9 Raw Authenticity"
- Neo-Minimalism = structural backbone. Clean layouts, generous whitespace, ONE bold moment per section. Type IS the composition in key sections.
- Emotion-Led = the soul. Hand-drawn micro-details, organic imperfections, visible craft. Every design layer injects empathy.
- Raw Authenticity = the texture. Intentional imperfection throughout. Grain, organic curves, slightly irregular spacing. "Someone made this with care, and you can tell."
- Brutalist (#2) = whisper only. Deliberate asymmetry in select layouts (hero, KOL grid). No systematic brutalism.
- Vibrant Color (#7) = reinterpreted within locked palette. Gradient layering (teal-to-darker-teal, merlot depth), color as emotional signal not decoration.

State explicitly: "This describes a REBUILD. The current page repeats centered-headline → subtitle → grid → CTA across every section. The rebuild breaks that pattern. Each section has a distinct compositional identity."

---

**Part 2: Global Design System**

These apply across ALL sections. Spec each precisely:

**Typography:**
- Display: PP Pangaia — KEEP. All 6 weights loaded (Ultralight 200, Medium 500, Bold 700, plus italic variants for each).
- Decision: PP Pangaia is MORE aligned with 2026 Emotion-Led direction than a high-contrast serif would be. Its geometric-humanist character has inherent warmth. The old 06-04's recommendation to switch to Canela/DM Serif is REVERSED.
- Body: Geist Sans (already loaded as --font-geist-sans)
- Scale: Define a typographic hierarchy:
  - Hero headline: PP Pangaia Ultralight (200), 108px-140px on xl, leading 0.92-0.95
  - Section headlines: PP Pangaia Bold (700), 48px-64px on lg, leading tight
  - Section sub-headlines: PP Pangaia Medium Italic (500i), 24px-32px, for emotional emphasis moments
  - Body: Geist Sans, 16px-18px, generous leading (1.6-1.7)
  - Eyebrow labels: Geist Sans, 11px-12px, tracking [0.25em], uppercase
  - Micro-detail: PP Pangaia Medium Italic at small sizes (14px-16px) for pull-quotes, attributions, inline emphasis

**Color system:**
- Palette LOCKED — do NOT change hex values:
  - Cloud White: #F5F4F2 (primary background)
  - Espresso: #1A1008 (primary text)
  - Deep Teal: #0D3D3A (brand primary, dark sections)
  - Merlot: #6B1E2E (emotional accent — CTAs, emphasis)
  - Warm Sand: #C8B89A (secondary, muted text alternative)
  - Burnished Amber: #C4892A (warm accent, decorative)
  - Border: #E8E4DF (dividers, card borders)
- Gradient usage (Trend #7 reinterpreted):
  - Deep teal sections: `linear-gradient(180deg, #0D3D3A 0%, #092D2B 50%, #0D3D3A 100%)` for depth
  - Merlot gradient moments: subtle merlot-to-darker for CTA hover states
  - Amber used only as small decorative accent — never as background
- Section rhythm: Alternate between Cloud White background sections and Deep Teal background sections. NOT every-other — create rhythm: Light, Light, Light, DARK, Light, Light, DARK, Dark. The dark sections are emotional peaks.

**Texture and craft elements:**
- Grain overlay: `.grain` CSS class (already exists in globals.css) — apply to sections that need tactile warmth. NOT every section. Use on: Hero, ArchetypePreview sample card, Footer.
- Hand-drawn SVG components (already exist in `components/ui/hand-drawn.tsx`):
  - HandDrawnDivider: replace mechanical `border-t` / `border-b` between content blocks
  - HandDrawnQuoteMark: use for testimonial quotes in SocialProof
  - HandDrawnCircle: decorative accent near eyebrow labels
  - EXPAND the library: add HandDrawnUnderline (wavy underline for emphasis words), HandDrawnArrow (organic arrow for CTA affordance)
- Organic border-radius: Already implemented as `organicRadius()` function in ArchetypePreview. Extend to other card-like elements. Each card should have slightly different border-radius values — NOT uniform rounded-2xl.

**Spacing:**
- Section padding: INCREASE from current py-24 to py-28 on mobile, py-36 on lg. Sections need to breathe.
- Inner content max-width: Vary per section. NOT always max-w-5xl centered. Some sections use max-w-3xl (intimate), some use max-w-6xl (expansive), some use asymmetric layouts.
- Vertical rhythm between elements: 1.5x-2x current spacing. Let content breathe.

**Motion:**
- ScrollReveal component (already exists): KEEP but tune parameters:
  - distance: 24px (already set)
  - duration: 800ms (already set)
  - easing: var(--ease-organic) = cubic-bezier(0.22, 1, 0.36, 1) (already set)
- Add stagger: elements within a section reveal with 100-150ms stagger delays
- Breathing animation: `.animate-breathe` (already exists) for decorative elements only
- NO parallax, NO scroll-hijacking, NO complex motion choreography
