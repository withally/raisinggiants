<task type="auto">
  <name>Task 1: Define color palette and typography direction</name>
  <files>.planning/phases/06-brand-positioning-research-and-visual-direction/06-04-visual-direction-brief.md</files>
  <action>
Read the positioning narrative (06-03-positioning-narrative.md) and reference brand gallery (06-02-reference-brand-gallery.md) first. Every recommendation must trace back to the positioning.

Create the visual direction brief with these sections:

**Positioning Anchor (brief)**
A 2-3 sentence summary of the positioning territory from Plan 03. This is the anchor all visual choices must trace to. Example: "The brand occupies the 'grounded self-discovery' territory — deep + warm, clinically credible but emotionally intimate. The visual identity must make someone feel 'this understands me' before they read a word."

**Color Palette Direction:**

1. **Current state assessment:** Document current palette (amber-50, stone-900, amber-600/700/400, white). Reference 06-RESEARCH.md assessment: "reads 'startup warm neutral' rather than 'emotionally distinctive.'"

2. **Evaluate the two paths** (from 06-RESEARCH.md):
   - Path A: Deepen and differentiate the existing amber palette (low disruption, preserves continuity)
   - Path B: Shift to a more distinctive chromatic identity (higher impact, more disruption)

   Make a recommendation based on what the positioning narrative establishes as the target emotional feeling.

3. **Recommended palette:** Specify 5-7 colors with:
   - Hex code
   - Name/description (e.g., "bone" not just "#F5F0E8")
   - Role (primary background, text, accent, highlight, muted)
   - Emotional rationale — WHY this color communicates the positioning territory

4. **Color relationships:** How do these colors work together? What's the visual rhythm? Which are dominant, which are accent?

5. **What to avoid:** Specific colors/palettes that would undermine the positioning (e.g., purple/clinical, neon/tech, pastel/wellness-generic).

**Typography Direction:**

1. **Current state assessment:** Cormorant Garamond (display) + Geist Sans (body). Reference 06-RESEARCH.md assessment: "directionally strong — editorial warmth + modern legibility."

2. **Display font evaluation:** Test Cormorant Garamond against the positioning:
   - Does it communicate "grounded self-discovery"?
   - Does it produce the right emotional register (intimate, deep, warm)?
   - Consider alternatives from 06-RESEARCH.md: Canela, Freight Display, Playfair Display
   - Make a keep/change recommendation with rationale

3. **Body font evaluation:** Test Geist Sans:
   - Does it pair well with the display font?
   - Is it readable enough for long-form result content?
   - Keep/change recommendation

4. **Type scale and treatment notes:** Any guidance on weight, size, spacing, or italic usage that reinforces the positioning.

Start the document with a frontmatter-style header (date, status, positioning anchor) and format for readability.

<task type="auto">
  <name>Task 2: Define imagery system, layout approach, and brand tone</name>
  <files>.planning/phases/06-brand-positioning-research-and-visual-direction/06-04-visual-direction-brief.md</files>
  <action>
Add the following sections to the visual direction brief:

**Imagery / Illustration System:**

1. **Current state:** Custom SVG illustrations (MirrorIllustration, ArchetypeIcons, KOL portrait sketches). Style is abstract/geometric.

2. **Direction recommendation:** Based on the positioning and reference brand patterns, recommend one of:
   - Character/illustration-based (relatable characters, metaphorical scenes)
   - Documentary/real people (candid photography, authenticity over polish)
   - Abstract/symbolic (shapes, textures, metaphors for internal states)
   - Hybrid (combination with rationale)

   Each recommendation must reference the positioning territory and cite evidence from the reference brand gallery.

3. **Illustration style notes:** If illustration-based, describe the style (line weight, color usage, level of detail, emotional register). Should illustrations feel hand-drawn, geometric, organic, editorial?

4. **Hero/key visual concept:** Describe the ideal hero visual for the landing page — what should it communicate? What visual metaphor serves the "mirror" concept? This is a concept brief, not a detailed spec.

5. **Photography guidelines (if applicable):** If the direction includes photography, describe: subject matter, lighting, color grading, composition approach.

**Layout Approach:**

1. **Spaciousness vs. density:** How much whitespace does this brand need? Reference the pattern synthesis.
2. **Editorial vs. app-like:** Where on the spectrum should the product sit?
3. **Section rhythm:** How should sections alternate (color, imagery, content density)?
4. **Key layout principle:** One overarching layout principle derived from the positioning (e.g., "generous negative space that creates room for reflection" or "intimate density that pulls the reader in").

**Brand Tone / Voice Direction:**

1. **Voice register:** Where on each spectrum:
   - Warm ←→ Clinical
   - Personal (you) ←→ Observational (one/they)
   - Poetic ←→ Plain
   - Confident ←→ Tentative

2. **Example copy transformations:** Take 2-3 current copy examples from the landing page and rewrite them in the recommended voice, showing the shift.

3. **Forbidden patterns:** Specific copy patterns that undermine the positioning (e.g., "parenting advice" framing, "take our quiz!" lead-magnet language, "transform your life" hyperbole).

**Mood Reference:**
A text-based mood summary that describes how all elements (color + type + imagery + layout + tone) work together. Describe a specific scenario: "A user lands on the page and sees [X]. The [color] background with [type] headline creates [feeling]. The [imagery] reinforces [message]. The overall effect is [emotional outcome]."

End the brief with a "Summary of Recommendations" table:
| Element | Current | Recommended | Change Level |
|---------|---------|-------------|--------------|
| Palette | amber/stone | [recommendation] | Low/Medium/High |
| Display font | Cormorant Garamond | [recommendation] | Keep/Change |
| Body font | Geist Sans | [recommendation] | Keep/Change |
| Imagery | Abstract SVG | [recommendation] | Low/Medium/High |
| Layout | Editorial warm | [recommendation] | Low/Medium/High |
| Brand name | Raising Giants | [from Plan 03] | Keep/Change |
