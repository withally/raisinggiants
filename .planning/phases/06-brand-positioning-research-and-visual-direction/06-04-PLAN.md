  <action>
Create the visual direction brief document. This REPLACES the existing 06-04-visual-direction-brief.md.

The brief must open with a one-paragraph summary of the positioning territory (from 06-03) to anchor all visual decisions.

**Section 1: Color Palette**

Define a complete color system with:

1. **Primary palette (3-4 colors):** These are the brand colors that appear on every product.
   - For each color: hex code, color name, emotional rationale (why this color serves the positioning territory), and usage context (where it appears — backgrounds, text, accents, CTAs)
   - Ground the palette in the reference brand patterns (06-02) and the RESEARCH.md color psychology findings
   - The RESEARCH.md directional finding: earthy warm palettes (terracotta, bone, warm charcoal) encode authenticity and groundedness; amber-based palette reads "startup warm neutral" — needs more distinctive chromatic identity

2. **Secondary/accent palette (2-3 colors):** These support the primary palette and enable product differentiation.
   - For each: hex code, name, rationale, usage context
   - Explain how accent colors can differentiate products within the brand family (e.g., The Mirror accent vs. The Blueprint accent) while maintaining brand coherence

3. **Semantic tokens:** Map colors to functional roles:
   - `--background`, `--foreground`, `--primary`, `--accent`, `--card`, `--muted`, `--destructive`
   - These map directly to the CSS custom properties in `globals.css`

4. **What the palette replaces:** Explicitly state what current palette elements are being replaced and why (the current amber/stone-900 system)

5. **Multi-product test:** For each primary color, explain why it works for The Mirror, The Blueprint, AND The Partner Match — not just one product

**Section 2: Typography**

Define the type system:

1. **Display font:** Name, weight(s), usage (headings, hero text, emotional emphasis). Include the rationale from RESEARCH.md: editorial serif fonts signal intellectual respect to millennial parents (per The New Yorker/Atlantic aesthetic).
   - If keeping Cormorant Garamond (RESEARCH.md recommends this): specify increased italic use for emotional emphasis as recommended
   - If recommending a change: provide full rationale and the new recommendation

2. **Body font:** Name, weight(s), usage. The current body font is Geist Sans — evaluate whether it serves the new positioning or needs to change.

3. **Type scale:** Recommended size relationships for headings, body, captions, CTAs.

4. **Typography rules:** How display and body fonts interact. Where each is used. What NOT to do (e.g., "never use display font for body text," "never use body font for hero headlines").

**CRITICAL: Every color and typography decision must trace back to the positioning territory.** The brief is not a mood board — it is a strategic document. Each decision should read: "We use [X] because the brand territory of [territory] requires [emotional quality] and [X] delivers that because [evidence from reference brands or color/type psychology]."

**Do NOT:**
- Build the palette around mirror/reflection metaphors (the brand is not The Mirror; The Mirror is one product)
- Use "startup template" warm neutrals without distinctive chromatic identity
- Recommend bright, playful, or child-coded colors (these signal child-facing products)

Append to the visual direction brief:

**Section 3: Imagery System**

Define the brand's visual language:

1. **Brand-level illustration direction:** What is the illustration style for the brand (not just for The Mirror)?
   - Reference the RESEARCH.md finding: the brand should encode "the parent's interiority — their psychological inner life" rather than a specific product metaphor
   - Specify the register: editorial line illustration, warm figurative, abstract-but-human, etc.
   - Name 2-3 illustrators or visual references whose work demonstrates the target register (e.g., School of Life, Kai Brammer, Olimpia Zagnoli — from RESEARCH.md)
   - Address the multi-product test: how does this illustration style serve The Mirror AND The Blueprint AND The Partner Match?

2. **Photography direction (if applicable):** Whether and how photography is used alongside illustration.
   - Subject guidance: adults as psychological beings, not "happy family" stock photos
   - Style guidance: warm, editorial, human — not clinical or corporate

3. **Product-specific illustration vs. brand illustration:** Define the distinction:
   - Brand illustrations: appear on landing page, brand materials, shared brand experiences — encode audience identity
   - Product illustrations: appear on individual product pages — can encode product metaphors (mirror for The Mirror, blueprint for The Blueprint)
   - This prevents the anti-pattern of building the brand visual identity around one product's metaphor

4. **What the imagery replaces:** Explicitly state what current illustration/imagery elements are being replaced (current geometric SVG icons, MirrorIllustration.tsx, ArchetypeIcons.tsx)

**Section 4: Layout Approach**

1. **Whitespace philosophy:** How much whitespace, and why (reference Kinfolk/editorial layouts if appropriate from reference brands)
2. **Content density:** How much content per viewport (especially for landing page and result page)
3. **Section rhythm:** How sections alternate (background colors, content width, visual weight)
4. **Responsive priorities:** Mobile-first considerations for the target audience

**Section 5: Brand Tone and Voice**

1. **Tone attributes (3-5):** e.g., warm, knowing, revelatory, grounded, peer-level. Each with a brief explanation and a do/don't example.
2. **Voice register:** Where does the brand sit on:
   - Clinical <---> Personal
   - Expert <---> Peer
   - Prescriptive <---> Revelatory
   - Serious <---> Light
3. **Vocabulary guidance:** Words the brand uses / Words the brand avoids. Reference the RESEARCH.md finding: "cycle-breaking" is cultural context, not brand vocabulary. The brand speaks in revelation, not therapy.
4. **Tagline direction:** If the positioning territory suggests a tagline or brand line, propose 3-5 options with rationale. If the emotional core from old research ("I always sensed this. Now I see it.") still works, it can be one option.

**Section 6: Multi-Product Visual System Summary**

This table proves the visual system is a brand system, not a single-product system.
